# Binarisation

* Исполняемый файл лежит в x64\Release\Binarisation.exe
Для запуска следует указать первым параметром путь к изображению 
вторым параметром путь, куда сохранять файл и для оценки скорости работы алгоритма
Третим параметром можно передать количество итераций для подсчета времени работы алгоритма.
* В папке x64\Release\Result лежат бинаризованный изображения двумя методами
алгоритм Бредли (хорошо описанный https://m.habr.com/ru/post/278435/) и обычный алгоритм с двумя трешхолдами(Easy_bin)
* Исходный код Binarisation\Binarisation\Source.cpp
* Алгоритм Бредли дает более хороший результат, устойчив к теням на фоне. Из этого достоинства логически вытекает недостаток метода Брэдли, а именно, плохая чувствительность к низкоконтрастным деталям изображения
* Бинаризация с двойным трешхолдом тоже дает хороший результат, но только на одноцветном тексте и без сильных шумов и теней.
* Время работы алгоритма записано в файле benchmark.txt
